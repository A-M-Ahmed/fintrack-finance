# FinTrack Client

React Frontend for FinTrack Personal Finance System.

## Tech Stack

- React (Vite)
- Tailwind CSS v4
- Shadcn/UI
- TanStack Query (Server State)
- Zustand (Auth State)
- Zod (Validation)
- Axios (API)
- React Router
- React Hook Form

## Setup

1. Copy `.env.example` to `.env`
2. Run `npm install`
3. Run `npm run dev`

## Pages

- `/signin` - Sign In
- `/signup` - Sign Up
- `/dashboard` - Main Dashboard
- `/wallets` - Wallet Management
- `/transactions` - Transaction List
- `/invoices` - Invoice Management
- `/settings` - User Settings
