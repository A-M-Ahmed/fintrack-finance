# FinTrack MERN Application

A comprehensive Personal Finance System built with the MERN stack (MongoDB, Express, React, Node.js).

## Quick Start

1. **Install Dependencies** (Root, Server, and Client):
   ```bash
   npm run install-all
   ```
   *Or manually install in each folder.*

2. **Run Development Server**:
   ```bash
   npm run dev
   ```
   This command starts both the **Node.js Backend** (Port 5000) and **React Frontend** (Port 5173).

## Directory Structure
- `/server`: Backend API and Database logic.
- `/client`: Frontend React application.

## Prerequisites
- Node.js installed.
- MongoDB running locally (default: `mongodb://localhost:27017/fintrack`).
